### How to run `docker compose` examples:
```sh
docker compose up
```
press `Ctrl-C` to terminate

### How to run `Dockerfile` examples:
#### Build
```sh
docker build -t \<img_name\> .
```
#### Run
```sh
docker run --rm \<img_name\>
```
press `Ctrl-C` to terminate
